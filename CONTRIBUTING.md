# অবদান রাখার গাইড (Contributing Guide)

ভল্ট প্রজেক্টে অবদান রাখতে চাওয়ার জন্য ধন্যবাদ! নিচের ধাপগুলো অনুসরণ করুন।

## কীভাবে শুরু করবেন

1. এই রিপোজিটরি **Fork** করুন
2. আপনার ফর্ক clone করুন:
   ```bash
   git clone https://github.com/<your-username>/<repo-name>.git
   cd <repo-name>
   ```
3. একটি নতুন ব্রাঞ্চ তৈরি করুন:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. `index.html` ব্রাউজারে খুলে সরাসরি পরিবর্তন টেস্ট করুন (কোনো বিল্ড স্টেপ নেই)
5. পরিবর্তন commit করুন:
   ```bash
   git commit -m "feat: সংক্ষিপ্ত বর্ণনা"
   ```
6. Push করে Pull Request খুলুন

## কোড স্টাইল

- **কোনো ফ্রেমওয়ার্ক/বিল্ড টুল নেই** — Vanilla JS, একটাই `index.html` ফাইলে রাখা হয়েছে
- Bengali UI টেক্সট বজায় রাখুন (এই প্রজেক্টের প্রধান audience বাংলাভাষী)
- নতুন ফিচার যোগ করলে নিশ্চিত করুন এটি ক্লায়েন্ট-সাইডেই থাকে — কোনো এক্সটার্নাল সার্ভারে ডেটা পাঠানো যাবে না
- এনক্রিপশন-সংক্রান্ত কোড পরিবর্তন করলে অবশ্যই ব্যাখ্যা দিন কেন এবং কীভাবে টেস্ট করেছেন

## বাগ রিপোর্ট

Issue খোলার সময় উল্লেখ করুন:
- কী ঘটেছিল বনাম কী প্রত্যাশা করেছিলেন
- ব্রাউজার ও ডিভাইসের তথ্য
- পুনরাবৃত্তির ধাপ (reproduction steps)

## ফিচার প্রস্তাব

নতুন ফিচারের আইডিয়া থাকলে আগে একটি Issue খুলে আলোচনা করুন, তারপর PR পাঠান — এতে দুই পক্ষেরই সময় বাঁচে।

## নিরাপত্তা সংক্রান্ত সমস্যা

এনক্রিপশন বা নিরাপত্তা দুর্বলতা খুঁজে পেলে পাবলিক Issue না খুলে সরাসরি মেইনটেইনারকে জানান। বিস্তারিত [docs/SECURITY.md](docs/SECURITY.md)-এ দেখুন।
