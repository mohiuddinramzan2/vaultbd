# ভল্ট (Vault) — অফলাইন এনক্রিপ্টেড পাসওয়ার্ড ম্যানেজার

**সম্পূর্ণ ক্লায়েন্ট-সাইড, সার্ভারবিহীন, ওপেন-সোর্স পাসওয়ার্ড ম্যানেজার ওয়েব অ্যাপ।**
কোনো ডেটা কখনো কোনো সার্ভারে পাঠানো হয় না — সবকিছু আপনার ব্রাউজারেই AES-256-GCM দিয়ে এনক্রিপ্ট হয়ে থাকে।

🔗 **লাইভ ডেমো:** GitHub Pages এ ডিপ্লয় করার পর এখানে লিংক যোগ করুন
`https://<your-username>.github.io/<repo-name>/`

## ✨ ফিচারসমূহ

- 🔐 **AES-256-GCM এনক্রিপশন** — PBKDF2 (250,000 iterations, SHA-256) দিয়ে মাস্টার পাসওয়ার্ড থেকে key derive করা হয়
- 📝 এন্ট্রি Add / Edit / Delete — সাইট, ইউজারনেম, পাসওয়ার্ড, লিঙ্ক, ক্যাটাগরি, নোট
- 🎲 কাস্টমাইজযোগ্য র‍্যান্ডম পাসওয়ার্ড জেনারেটর
- 📊 পাসওয়ার্ড স্ট্রেংথ মিটার (এনট্রপি ভিত্তিক)
- 🔍 সার্চ ও ক্যাটাগরি অনুযায়ী ফিল্টার
- 📋 এক-ট্যাপে কপি (ক্লিপবোর্ড), শো/হাইড টগল
- ⏱️ ৫ মিনিট নিষ্ক্রিয়তায় অটো-লক
- 🔑 মাস্টার পাসওয়ার্ড পরিবর্তন (পুরো ভল্ট রি-এনক্রিপ্ট হয়)
- 💾 এনক্রিপ্টেড ব্যাকআপ এক্সপোর্ট/ইমপোর্ট (টেক্সট ফরম্যাটে, ম্যানুয়াল ব্যাকআপের জন্য)
- 📱 মোবাইল-ফ্রেন্ডলি রেসপন্সিভ ডিজাইন

## 🚀 ব্যবহার শুরু করবেন কীভাবে

### অপশন ১: সরাসরি ব্যবহার
`index.html` ফাইলটি যেকোনো আধুনিক ব্রাউজারে খুলুন — কোনো ইনস্টলেশন লাগবে না, কোনো সার্ভার লাগবে না।

### অপশন ২: GitHub Pages এ হোস্ট করুন
1. এই রিপোজিটরি GitHub-এ push করুন
2. **Settings → Pages → Source: main branch, /root** সিলেক্ট করুন
3. কিছুক্ষণ পর `https://<username>.github.io/<repo>/` এ লাইভ হয়ে যাবে

### অপশন ৩: লোকাল সার্ভার (ঐচ্ছিক)
```bash
git clone https://github.com/<username>/<repo-name>.git
cd <repo-name>
python3 -m http.server 8080
# ব্রাউজারে খুলুন: http://localhost:8080
```

## ⚠️ গুরুত্বপূর্ণ সতর্কতা

- **মাস্টার পাসওয়ার্ড হারালে ডেটা পুনরুদ্ধারের কোনো উপায় নেই।** এটাই প্রকৃত end-to-end এনক্রিপশনের নিয়ম — কোনো "ব্যাকডোর" নেই।
- ডেটা ব্রাউজারের `localStorage`-এ **শুধু সেই ডিভাইস/ব্রাউজারেই** সংরক্ষিত থাকে। একাধিক ডিভাইসে ব্যবহার করতে চাইলে "এনক্রিপ্টেড ব্যাকআপ" ফিচার দিয়ে ম্যানুয়ালি সিঙ্ক করুন।
- ব্রাউজার ক্যাশ/ডেটা মুছে ফেললে ভল্ট হারিয়ে যাবে — নিয়মিত ব্যাকআপ নিন।
- এটি একটি ব্যক্তিগত/শিক্ষামূলক প্রজেক্ট; উচ্চ-ঝুঁকির ব্যবহারের জন্য প্রতিষ্ঠিত অডিটেড পাসওয়ার্ড ম্যানেজার ব্যবহার বিবেচনা করুন।

আরও প্রযুক্তিগত বিস্তারিত জানতে দেখুন [docs/SECURITY.md](docs/SECURITY.md) এবং [docs/FEATURES.md](docs/FEATURES.md)।
